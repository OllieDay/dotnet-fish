# dotnet-fish
[fish shell](https://fishshell.com/) alias for the `dotnet` command.

## Installation
Copy `dn.fish` to `~/.config/fish/functions`

## Usage
| dn     | dotnet         |
| ------ | -------------- |
| `dn`   | `dotnet`       |
| `dn c` | `dotnet clean` |
| `dn b` | `dotnet build` |
| `dn r` | `dotnet run`   |
| `dn t` | `dotnet test`  |
